# AcousticMaterial class

Defines how a surface interacts with sound.

Supports defining:

`Absorption` - the fraction of sound energy absorbed on reflection. Used by reverb and room estimation.

`1.0` = full absorption (less reverb). `0.0` = no absorption (full reverb).

`Scattering` - how diffusely the surface reflects sound.

`0.0` = perfect mirror. `1.0` = fully diffuse.

`Transmission` - fraction of sound energy that passed through the surface. Used by occlusion.

`0.0` = fully blocked. `1.0` = fully transparent.

Material uses three band model for frequency dependent behavior.

`Low ≤ 400 Hz``Mid = 400-2500 Hz``High ≥ 2500 Hz`
