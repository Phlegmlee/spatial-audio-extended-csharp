# SpatialAudioPlayer3D.IgnoreListenerBody property

When enabled, the listener's `CharacterBody3D` (if any) is automatically excluded from occlusion raycasts. This prevents the player's own collision shapes from being detected as walls.

Detection walks up the scene tree from the active `Camera3D` looking for the first `CharacterBody3D` ancestor, then excludes it and all of its collision shape children.
