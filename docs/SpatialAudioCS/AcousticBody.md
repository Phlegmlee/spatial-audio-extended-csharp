# AcousticBody class

Attach as a child to any `CollisionObject3D` to give the surface acoustic properties using an [`AcousticMaterial`](./AcousticMaterial.md).

Supported Parent Nodes:

* `StaticBody3D` - static level geometry, walls, floors, props, etc.
* `RigidBody3D` - dynamic movable objects.
* `CharacterBody3D` - player or npc bodies
* `Area3D` - trigger zones / volumes that affect sound, water, fog etc.
* `CSGShape3D` - must have collision enabled.

The AcousticBody must be a direct child of parent node's collison object.

TIP: When a compatable node is selected an editor button is created in same location as the button when a mesh is selected.

```csharp
public class AcousticBody : Node
```
