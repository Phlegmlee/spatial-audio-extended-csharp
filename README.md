# Spatial Audio Extended
![Static Badge](https://img.shields.io/badge/REWRITE_STATUS-IN_PROGRESS_55%25-orange)
[![maintained](https://img.shields.io/badge/Status-REWRITE-red.svg)](CHANGES.md)

[![documentation](https://img.shields.io/badge/Documentation-yellow)](https://phlegmlee.github.io/godot-portals-plugin-csharp/)


Spatial Audio Extended provides an advanced, drop-in replacement, SpatialAudioPlayer3D, for Godot's AudioStreamPlayer3D. Originally created by Danikakes, it provides physically-inspired, real-time spatial audio effects for immersive 3D games. 


## Documentation
Visit the documentation site [here](https://phlegmlee.github.io/spatial-audio-extended-csharp/).
