# spatial-audio-player-3d-csharp
SpatialAudioPlayer3D is an advanced, drop-in replacement for Godot's AudioStreamPlayer3D. Originally created by Danikakes, it provides physically-inspired, real-time spatial audio effects for immersive 3D games. 

_Note: This is a C# rewrite of [spatial_audio_player_3d](https://github.com/Danikakes/spatial_audio_player_3d) by Danikakes.
